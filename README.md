# Аналитический кейс: факторы, влияющие на доход пользователей

## Контекст
Цель проекта — проанализировать факторы, влияющие на уровень дохода пользователей, 
и проверить ряд гипотез с использованием статистических методов и регрессионного анализа.

## Данные
Используются данные обследования РМЭЗ НИУ ВШЭ (2024), репрезентативная выборка по индивидам.
После фильтрации и очистки данных сформирована аналитическая выборка.

## Гипотезы
- Уровень образования влияет на доход
- Семейное положение связано с доходом
- Опыт работы оказывает влияние на доход
- Продолжительность рабочего дня связана с доходом

## Анализ
- Предобработка и очистка данных
- Описательная статистика
- Проверка гипотез
- Построение регрессионных моделей
- Анализ качества моделей и ограничений данных

## Основные выводы
- Подтверждено влияние семейного положения на доход
- Большинство факторов оказались статистически незначимыми на данной выборке
- Малый размер выборки существенно ограничивает интерпретацию результатов

## Ограничения и дальнейшие шаги
- Небольшой объем данных
- Возможная мультиколлинеарность
- Для более устойчивых выводов требуется расширение выборки

## Инструменты
Python, pandas, statsmodels, matplotlib
